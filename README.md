# challenge-amigo-secreto
Challenge del amigo secreto para demostrar los conocimientos obtenidos en el curso de logica de programación
Para poder abrir y correr este archivo, es necesario abrir el archivo "Index.html"
<img width="455" alt="image" src="https://github.com/user-attachments/assets/ab63cf11-b593-4f46-8c5d-dc3cc1894df9" />
Una vez abierto, te abrirá la página principal, donde deberás ingresar el nombre de amigo que te gustaría añadir a la lista en el field "Esceribe un nombre"
<img width="953" alt="image" src="https://github.com/user-attachments/assets/9998e331-38a8-4880-b3ab-36624921aa25" />
El siguiente paso es seleccionar "Añadir", y por consiguiente el nombre será impreso en la pantalla
<img width="494" alt="image" src="https://github.com/user-attachments/assets/3c4da4a2-91ce-430e-8948-8c3058151661" />
Tendrás que añadir más nombres, no hay limite 
<img width="502" alt="image" src="https://github.com/user-attachments/assets/301555d2-6933-422c-8140-b3a22f103bf4" />
Una vez añadidos los nombres, selecciona "Sortear amigo", y la pantalla imprimirá el nombre del amigo secreto
<img width="535" alt="image" src="https://github.com/user-attachments/assets/63a4bc66-eb36-411f-8bb3-92f65562544d" />
Listo, así es como funciona el programa de "Adivinar el amigo secreto"

Nota: Si no se añade ningún nombre, la página imprimirá el mensaje "Por favor añade un nombre"
